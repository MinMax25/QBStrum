インストール・アンインストール手順

以下の手順でインストールおよびアンインストールを行ってください。

  • https://github.com/MinMax25/QBStrum/releases/tag/v1.0.0
  • インストール
    1. 上記のリンクより最新のリリース版ZIPファイルをダウンロードします
    2. ダウンロードした zip を任意の場所に展開します
    3. 展開されたフォルダ内の QBStrum.vst3 をフォルダごと VST3 プラグインフォルダへ配置します
      → C:\Program Files\Common Files\VST3
    4. 展開されたフォルダ内の preset フォルダ内のファイルをそれぞれのフォルダへ配置します
      1. default.qbs
        → C:\Users\（ユーザー名）\Documents\QBStrum\Presets\
      2. QBStrum.pitchlist
        → Studio One ユーザープリセットフォルダ\Pitch Names\
      3. QBStrum.keyswitch
        → Studio One ユーザープリセットフォルダ\Key Switches\
  • アンインストール
    • 不要になった場合はフォルダごと削除してください（レジストリは使用していません）
