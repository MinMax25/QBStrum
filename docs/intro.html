はじめに

Studio One（Fender Studio Pro）で使用するギター用ストラム／コード演奏支援 VST3 プラグインです

システム要件

動作環境はWindows OSのみのとなります。

Studio One Pro Windowsシステムの要件に準拠します。

概要

QBStrum は、コード指定 + ストラム／ブラッシング操作をMIDI レベルで制御するギター演奏支援プラグインです。
コード、ボイシング、ストラム方向、ブラシ、ミュート、アルペジオ、アーティキュレーションなどを
KeySwitch とパラメータ操作で統合的に扱えます。
※ 本プラグインは MIDI 処理専用であり、オーディオ信号は生成しません。
