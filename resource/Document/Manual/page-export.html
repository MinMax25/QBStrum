エクスポート

ドラムマップのエクスポートを行います。

使い方

画面右のキットリストよりエクスポート対象を選択（複数選択可）、 希望するエクスポートボタンを押します。 表示されたフォルダ選択ダイアログで格納先を選択します。 選択したフォルダは設定ファイルに記録され、次回エクスポートの際に初期選択フォルダとなります。

アーティキュレーションの使用

Use Articulation Pitch Nameにチェックがある場合、ピッチ名にはアーティキュレーション名が優先的に使用されます。

通常のドラムマップエクスポート時は Use Articulation When Exportingでアーティキュレーションの使用する・しないを指定できます。

アーティキュレーションの使用しないを選択した場合、ピッチリストの表示内容がそのまま出力されます。

アーティキュレーションの使用するを選択した場合、ピッチリストの表示内容ではなく紐づけされたアーティキュレーションによって並び順が決定されます。

ドラムスコア情報や配色はアーティキュレーションの使用するを選択した場合のみ出力されます。（Cubaseの配色情報はありません）

ファイル名について

ファイル名にはキット名.拡張子となります。すでに同名ファイルが存在する場合は上書き置換されます。

Studio One用ピッチリスト、Cubase用ドラムマップ、tsvテキストはプラグイン名フォルダ配下に出力されます。

ファイル種別

エクスポート可能なファイルは以下の通りです。

  • 通常のドラムマップ
    • Studio One用ピッチリスト
    • Cubase用ドラムマップ
  • マッピングされたドラムマップ
    • Cubase用ドラムマップ
    • Note Mapper用テンプレート
  • その他
    • tsvテキスト

マッピングされたドラムマップをエクスポートする場合は元になるドラムマップ（キット）を選択してください。
