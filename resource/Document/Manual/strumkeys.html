Strum Keys

ストラム操作に割り当てているキースイッチを設定します。
ここで割り当てたキースイッチはQBStrum.pitchlistに定義してあります。

機能 Pitch Note 説明
All Notes Off 53 F2 すべての発音中ノートを強制的に停止します
Brush Down 63 D#3 下方向にブラッシングします
Brush Up 61 C#3 上方向にブラッシングします
Up High 64 E3 高音弦のアップストローク
Up 62 D3 全弦のアップストローク
Up Low 60 C3 低音弦のアップストローク
Down High 59 B2 高音弦のダウンストローク
Down 57 A2 全弦のダウンストローク
Down Low 55 G2 低音弦のダウンストローク
Mute 1 58 A#2 音源のミュート音を発音します
Mute 2 56 G#2 音源のミュート音を発音します
String 1 52 E2 1弦のみを指定して発音させます
String 2 50 D2 2弦のみを指定して発音させます
String 3 48 C2 3弦のみを指定して発音させます
String 4 47 B1 4弦のみを指定して発音させます
String 5 45 A1 5弦のみを指定して発音させます
String 6 43 G1 6弦のみを指定して発音させます
  • ノート右のボタンでDAWにストラムトリガーを送信します。左側＞右側でヴェロシティ値が大きくなります。
  • 割り当てられているピッチは変更可ですが、その場合はDAWに取り込むピッチリストも併せて修正する必要があります。
