# QBStrum

**QBStrum** は、DAW 上でギターのストラム・アルペジオ演奏を  
**直感的かつ演奏的にコントロールするための MIDI 制御プラグイン**です。  
Studio One のキースイッチ＆Pitch Name UI と高い親和性を持ち、  
Fender Studio Pro などのギター音源と組み合わせて使うことで、  
**“弾き方を指定して演奏させる”** ギターワークフローを実現します。

## 🧠 概要

- ギター音源ではなく **MIDI 制御エンジン**
- ストラム / アルペジオ / ミュート指定が可能
- Studio One のキースイッチ・Pitch Name List との相性が抜群
- Fender Studio Pro など外部ギター音源と組み合わせて使用

---

## ⚙️ システム要件

- DAW：Studio One / Cubase / 他 MIDI ノート・CC が扱える DAW
- MIDI 対応ギター音源（例：Fender Studio Pro）
- VST3 対応ホスト
- Windows / macOS（対応 OS を明記）

---

## 🧩 特徴

### 🎸 ストラム & アルペジオ指定

ノート番号ではなく **“意味”** として演奏を指定できます：

- Down / Down High / Down Low
- Up / Up High
- Brush
- All Mute
- Mute 1 / Mute 2
- Arpeggio（弦 1〜6）  
（Studio One の PitchNameList を前提とした設計）

---

### 🎼 大量のコードプリセット

QBStrum は、**1,044 種類のコードプリセット**を内蔵しています：

- キーノート：12（C ～ B）
- コードタイプ：29（Major / minor / 7 / sus など）
- ポジション：3（異なるボイシング）

さらに **ユーザー編集用 120 スロット**があり、  
独自のコード・ボイシングを保存できます。

---

### 🎛 フレット・ポジション制御

- 各弦ごとに **-4 ～ +4 フレット + Open** オフセット指定
- 基本ポジションとオフセット位置を UI 上で色分け表示
- 実演奏に近いニュアンスコントロール

---

## 🚀 使い方（簡単ステップ）

1. DAW 上でトラックに QBStrum を挿す  
2. Fender Studio Pro などの音源をインサート  
3. キースイッチを DAW 側で配置  
4. コード指定 / ストラム方向 / オフセットを調整  
5. 再生して演奏を確認

---

## 💡 Studio One でのおすすめ設定

Studio One では：

- **KeySwitch List** を利用して演奏タイプを整理
- **Pitch Name List** でノートを視覚化
- CC98 / CC99 でコード指定

など、**QBStrum の思想と UI が噛み合う**ので  
最も快適に使うことができます。

---

## ⚠️ 向いていない用途（正直）

QBStrum は **演奏設計・制御を重視**したツールです。  
したがって、以下の用途には適しません：

- プラグイン単体で完結するギター音源
- 自動的に最適なギターフレーズを生成してくれるツール
- セット＆忘れのループ生成器

代わりに、フレーズの **細かい設計・編集**をしたい人に適しています。

---

## 🧠 よくある質問（FAQ）

### ▶ Cubase でも使えますか？

はい。Cubase の Expression Map で  
CC98 / CC99 をコード指定として送ることで対応できます。

ただし、Studio One の UI（Pitch Name / KeySwitch）ほど  
統合された表示体験は得られません。
